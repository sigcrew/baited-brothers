# 대한민국 해안 경계

`korea-adm0.json`은 GeoJSON 형식으로 저장한 geoBoundaries의 대한민국 ADM0 경계 고정 버전입니다.

- 원본: Natural Earth / geoBoundaries
- 경계 연도: 2021
- 빌드: geoBoundaries 2023-12-12
- 라이선스: Public Domain
- API 메타데이터: https://www.geoboundaries.org/api/current/gbOpen/KOR/ADM0/
- 고정 원본: https://github.com/wmgeolab/geoBoundaries/tree/9469f09/releaseData/gbOpen/KOR/ADM0

지도에서 선택한 육지 좌표가 해안선으로부터 3km 이내인지 오프라인으로 판정하는 데 사용합니다.
