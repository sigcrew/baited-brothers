# 지도 (후순위)

> Polarsteps처럼 **내가 잡은 곳 + 사진**을 지도에서 보는 기능.  
> 공공 낚시 포인트 지도가 아니라 **개인 출조·조과 일기 지도**.

관련: [`collection-verification.md`](./collection-verification.md) · [`navigation.md`](./navigation.md) · [`positioning-monetization.md`](./positioning-monetization.md)

---

## 1. 한 줄

> 인증 조과의 셔터 GPS를 핀으로 찍고, 그 자리에서 사진을 확인한다.  
> **내 조과만** 보인다.

---

## 2. 왜 맞는지

| 포인트 | 설명 |
|--------|------|
| 브랜드 | 현장 증빙·수집 감성과 맞음 |
| 데이터 | 수집 인증의 셔터 GPS·사진과 직결 |
| 차별 | 도감·일정만의 유틸을 “출조 스토리”로 올림 |
| 공유 | 조과 카드 다음 단계의 자랑 축 |

---

## 3. 하지 않는 것

| 제외 | 이유 |
|------|------|
| 전국 낚시터/조황 공공 지도 | 물반고기반 등과 정면 경쟁, 공급 비용 큼 |
| 하단 **지도 전용 탭** (초기) | 홈·수집·프로필 + 기록하기 CTA로 이미 충분 |
| 조과 없이 빈 지도 우선 출시 | Polarsteps 느낌이 안 남 → 수집 루프 먼저 |

---

## 4. 넣는 시점

| 단계 | 내용 |
|------|------|
| **지금** | 미구현. 카메라·GPS·카드 앨범 우선 |
| **다음** | 수집 > 카드에 **「지도로 보기」** 토글/버튼 |
| **이후** | 홈 서브 진입 또는 필요 시 탭 재검토 |

완료한 출조 일정(`fishing_trips` done)과 조과 핀을 이으면 “그날 어디서 뭘 잡았는지” 스토리가 된다. (후순위 연동)

---

## 5. UX 스케치 (나중)

```
수집 > 카드
  → [목록] | [지도]
  → 핀 = 인증 조과 (GPS 있는 건만)
  → 핀 탭 = 사진·어종·날짜·조과 카드
```

미인증·GPS 없는 기록은 지도에 안 올리거나 약한 표시.
