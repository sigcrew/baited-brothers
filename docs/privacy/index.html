개인정보 처리방침

BAITED BROTHERS · EFFECTIVE 2026.07.21

1. 처리하는 정보

계정 이메일과 로그인 제공자 정보, 사용자가 작성한 출조·조과·메모, 촬영 사진, 조과 인증을 위한 촬영 시점의 위치 정보, 앱 버전·운영체제와 기능 이용 기록(AI 처리 결과, 도감·카드 열람, 출조·조과 변경)을 처리합니다. 기능 이용 기록에는 사진, 좌표, 메모, 이메일을 중복 저장하지 않습니다.

2. 이용 목적

로그인과 계정 관리, 출조·조과 기록 보관, 도감 해금, 사진 기반 어종 후보 추천, 오류 대응, 서비스 안정성·사용 흐름 개선, AI 및 저장 용량 운영에 사용합니다.

3. 외부 처리자

  • Supabase: 인증, 데이터베이스, 사진 및 기능 이용 기록 저장
  • Anthropic: 사용자가 판별을 요청한 사진과 폐쇄형 도감 후보를 이용한 어종 후보 추천
  • Apple·Google: 사용자가 선택한 소셜 로그인 인증

AI 추천은 보조 정보이며 최종 어종은 사용자가 직접 확인합니다.

4. 보관과 삭제

계정이 유지되는 동안 출조·조과 기록과 업로드 사진을 보관합니다. 원본 기능 이용 기록은 최대 90일 후 삭제하며, 이후에는 사용자를 식별할 수 없는 기간별 집계만 남길 수 있습니다. 프로필의 계정 탈퇴를 사용하면 계정과 연결된 출조·조과 기록, 업로드 사진 및 원본 이용 기록을 삭제합니다. 법령상 보관 의무가 있는 경우 해당 기간 동안 별도로 보관할 수 있습니다.

5. 사용자 권리

사용자는 앱에서 조과 기록을 수정·삭제하고 계정을 탈퇴할 수 있습니다. 문의나 권리 행사는 아래 공개 지원 채널을 통해 요청할 수 있습니다.

6. 위치·사진 권한

카메라는 조과 촬영에, 사진 보관함은 사용자가 선택한 사진에, 위치는 촬영 시점 조과 인증에만 사용합니다. 사용자는 기기 설정에서 언제든 권한을 철회할 수 있습니다.
