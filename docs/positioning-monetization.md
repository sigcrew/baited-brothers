# 수집형 도감 포지셔닝 & 수익화

> Baited Brothers (낚시당한 녀석들)  
> 기준 포지션: **수집형 도감** — 잡은 물고기를 도감에 모으고, 시즌마다 완성도를 자랑하는 앱

---

## 1. 한 줄 포지션

> 초보·중급 낚시꾼이 현장에서 어종을 확인하고, 잡은 녀석을 도감에 모아 시즌마다 완성도를 자랑하는 앱

---

## 2. 포지셔닝 비교 (참고)

| | 1. 규제 도우미 | 2. 수집형 도감 ✅ | 3. 예약 브릿지 |
|--|----------------|------------------|----------------|
| 타깃 | 입문~초보 | 기록·수집 좋아하는 중급자 | 갈 곳 모르는 초보 + 제휴처 |
| 핵심 가치 | 방생·금어기 10초 확인 | 도감 완성·뱃지·시즌 챌린지 | 추천 포인트 → 예약 연결 |
| MVP 적합도 | 높음 | 높음 | 낮음 (영업 필요) |
| 차별화 | 중~높 | 높 | 중 (경쟁 심함) |
| 브랜드 핏 | 보통 | **가장 잘 맞음** | 약함 |

**채택:** 2를 코어로 두고, 1(규제/최소크기 정보)은 무료 기본 가치로 유지.

---

## 3. 수익화 원칙

| 원칙 | 의미 |
|------|------|
| 기록은 무료 | 잡기·저장·기본 도감 채우기는 막지 않음 |
| 자랑·완성은 유료 | 스킨, 시즌, 심화 통계만 잠금 |
| 페이월은 “예쁨/속도/깊이” | 핵심 기능 자체를 막지 않음 |

**한 줄:** 잡은 기록은 공짜, “더 예쁘게·더 완전하게·시즌마다 새로”는 유료.

---

## 4. 수익 모델

### 4-1. 시즌 패스 / 배틀패스 (메인)

- 시즌마다 챌린지·뱃지·테마 잠금 해제
- 예: “여름 연안 시즌”, “갈치 시즌”
- 가격 감: **시즌당 4,900~9,900원**

### 4-2. 코스메틱 (보조)

- 도감 프레임, 프로필 테마, 뱃지, 조과 카드 스킨
- 어종·기록 잠금 금지 → **꾸미기만** 유료

### 4-3. 프리미엄 구독 (헤비 유저)

- 무제한 고화질 사진, 상세 통계, 희귀도/완성도 분석
- “못 채운 어종 TOP”, 시즌별 진행률
- 가격 감: **월 2,900~4,900원** 또는 연간

### 4-4. 디지털 컬렉터블 (선택)

- 이벤트 한정 뱃지, 대회·브랜드 콜라보
- NFT까지 갈 필요 없음 → 앱 내 한정 아이템으로 충분

### 4-5. 광고 (약하게)

- 도감 목록 하단·공유 전 정도만
- 수집 앱은 감정형 → 광고 과다 시 브랜드 훼손

### 추천 레이어 조합

| 레이어 | 모델 | 역할 |
|--------|------|------|
| 코어 | 무료 도감 + 기록 | 유입·리텐션 |
| 메인 수익 | 시즌 패스 | 반복 결제 |
| 보조 | 스킨/뱃지 | ARPU 상승 |
| 선택 | 연간 프리미엄 | 헤비 유저 |

## 5. 게이미피케이션 범위

> 게이미피케이션은 O. 다만 “게임 앱”이 아니라 **도감을 완성하는 재미** 수준으로.

수집형 도감 포지션에서 게이미피케이션은 옵션이 아니라 **리텐션 엔진**이다.  
과하면 “유치하다 / 현질 유도”로 브랜드가 깨지므로, **라이트 3단**만 채택한다.

### 5-1. 원칙

| 원칙 | 의미 |
|------|------|
| 모으고 싶게 | 게임처럼 보이게 만들지 않음 |
| 기록은 막지 않음 | 에너지·행동권·횟수 제한 금지 |
| 수익과 연결 | 시즌 패스·스킨을 사게 만드는 이유가 되게 |
| 낚시 감성 유지 | XP·레벨 범벅, 가챠는 배제 |

### 5-2. 할 것 (채택)

| 요소 | 목적 | 무료/유료 |
|------|------|-----------|
| 도감 완성도 % | 핵심 수집 루프 | 무료 (기본) |
| 카테고리·시즌 완성 | 중간 목표 제공 | 무료 / 시즌 심화는 패스 |
| 시즌 챌린지·뱃지 | 재방문 + 패스 판매 | 기본 뱃지 무료, 시즌 뱃지 패스 |
| 희귀도·첫 포획 표시 | “자랑” 감정 | 기본 무료, 심화 힌트는 프리미엄 |
| 연속 출조(스트릭) | 재방문 유도 (과장 금지) | 무료, 알림은 최소 |
| 친구와 완성도 비교 | 소셜·공유 촉진 | 무료 (선택 기능) |
| 조과 카드 꾸미기 | 자랑 UX + 코스메틱 매출 | 기본 1종 무료, 스킨 유료 |

### 5-3. 안 할 것 (제외 또는 매우 늦게)

| 요소 | 이유 |
|------|------|
| 경험치·레벨업 범벅 | 캐주얼 게임처럼 보임, 낚시 감성 훼손 |
| 에너지 / 행동권 제한 | 기록 무료 원칙과 충돌 |
| 전체 서버 랭킹 과열 | 조작·부정 기록 유인 |
| 가챠성 박스 | 신뢰·브랜드 훼손 |
| 일일 퀘스트 도배 | 알림 피로 → 앱 삭제 |

### 5-4. 라이트 게이미피케이션 3단

1. **수집** — 어종 해금, 진행률, 카테고리 완성  
2. **시즌** — 기간 한정 챌린지 + 뱃지 (시즌 패스와 연결)  
3. **자랑** — 예쁜 조과 카드, 친구 비교 (선택)

### 5-5. 수익화와의 연결

| 게이미피케이션 | 연결 상품 |
|----------------|-----------|
| 시즌 챌린지·뱃지 | 시즌 패스 |
| 조과 카드·도감 프레임 | 코스메틱 / 패스 스킨 |
| 심화 완성 가이드·희귀도 | 프리미엄 구독 |
| 기본 진행률·기록 | 항상 무료 (신뢰) |

## 6. 무료 / 유료 기능 경계표

| 영역 | 무료 | 시즌 패스 | 프리미엄 (구독) |
|------|------|-----------|-----------------|
| **도감 열람** | 전체 어종·카테고리·설명·최소크기 | 시즌 한정 도감 테마/프레임 | 동일 + 희귀도·완성 힌트 |
| **낚시 기록** | 무제한 기록 (어종·크기·날짜·메모) | — | — |
| **사진** | 장당 1장, 기본 압축 | — | 고화질·다중 사진 |
| **도감 진행률** | 기본 % (잡은 종/전체) | 시즌별 진행률·챌린지 트랙 | 지역·시즌·카테고리 심화 분석 |
| **뱃지** | 기본 뱃지 소수 | 시즌 뱃지·챌린지 보상 | 과거 시즌 뱃지 아카이브 |
| **꾸미기** | 기본 카드 1종 | 시즌 스킨·프레임 | 전체 스킨 라이브러리 |
| **공유** | 기본 조과 카드 | 시즌 한정 공유 템플릿 | 워터마크 제거·커스텀 문구 |
| **통계** | 최근 N건, 간단 요약 | 시즌 랭킹 (선택) | 월/연 리포트, “못 채운 어종” |
| **알림** | 없음 또는 최소 | 시즌 챌린지 알림 | 금어기·시즌 시작 알림 |
| **광고** | 가벼운 배너 가능 | 시즌 중 광고 축소 | 광고 제거 |

## 7. 절대 유료화하지 말 것

- 어종 도감 **열람 잠금**
- 기록 **횟수 제한**
- 방생/최소크기 등 **안전·규제 정보**
- 기본 로그인·동기화

## 8. 유료로 두기 좋은 것

- 시즌 챌린지·뱃지
- 조과 카드/도감 스킨
- 고화질·다중 사진
- 심화 통계·완성 가이드
- 광고 제거

## 10. 소셜 범위

> 풀 커뮤니티(피드·게시판·댓글)는 **하지 않는다.**  
> 범위는 **공유 + 친구 비교**까지.

| 포함 | 설명 |
|------|------|
| **조과 카드 공유** | 인증 기록을 이미지/링크로 외부 SNS·메시지에 공유 |
| **친구와 도감 비교** | 초대한 친구끼리 완성도·시즌 진행 비교 (선택 기능) |

| 제외 (의도적) | 이유 |
|----------------|------|
| 전체 타임라인 / 조황 게시판 | 경쟁사 강세, 운영 비용 큼 |
| 댓글·좋아요 중심 피드 | 핵심 루프(수집)와 거리 있음 |
| 지역 오픈 채팅 | 모더레이션 부담 |
| 전체 서버 공개 랭킹 | 조작 인센티브·분쟁 (시즌·친구 단위만 허용 가능) |

게이미피케이션 ‘자랑’ 단과 동일 선상이다. 리텐션이 정체되고 “자랑할 상대가 없다”는 신호가 나오기 전에는 게시판형 커뮤니티로 확장하지 않는다.
