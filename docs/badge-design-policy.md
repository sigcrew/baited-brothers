# 뱃지 디자인 정책

## 목적

수집 화면의 뱃지는 작은 크기에서도 해금 조건과 카테고리가 구분되는 자수 와펜으로 제작한다. 새 뱃지는 기존 에셋의 색상, 재질, 외곽 형태 체계를 유지한다.

## 카테고리별 외곽 형태

| 카테고리 | 의미 | 기본 외곽 형태 | 사용 원칙 |
| --- | --- | --- | --- |
| `VOYAGE` | 출조, 항로, 장소 | 하단이 뾰족한 방패형 | 항해·이동·장소와 관련된 조건에만 사용한다. |
| `FIELD NOTE` | 일지, 시간대, 현장 조건 | 팔각형 또는 가로형 기록 패치 | 시간·환경은 팔각형, 기록 작성 행위는 가로형 패치를 사용한다. |
| `CATCH` | 조과, 크기, 연속 기록 | 원형 | 물고기를 잡은 결과와 크기 기록은 모두 원형으로 통일한다. |
| `SPECIMEN` | 어종 수집, 도감 진행 | 육각형 | 여러 어종, 관찰, 분류와 관련된 조건에 사용한다. |

카테고리 경계가 애매하면 사용자가 뱃지를 획득하게 만든 핵심 행동을 기준으로 정한다. 예를 들어 대어 기록은 크기 측정이 포함되더라도 핵심 행동이 조과이므로 `CATCH` 원형을 사용한다.

## 시각 스타일

- 실제 천 위에 촘촘히 놓인 프리미엄 자수 와펜 질감을 사용한다.
- 굵은 외곽 자수, 얇은 아이보리 안쪽 테두리, 단순한 중앙 상징의 3단 구조를 기본으로 한다.
- 금속 핀, 에나멜, 플라스틱 광택처럼 보이는 표현은 사용하지 않는다.
- 입체감은 실의 결, 겹친 자수, 가장자리 두께로만 표현한다.
- 중앙 상징은 148px 표시에서도 즉시 알아볼 수 있도록 큰 덩어리 위주로 구성한다.
- 작은 글자와 장식용 문장은 이미지 안에 넣지 않는다.

## 색상

| 역할 | 색상 |
| --- | --- |
| 바탕과 가장 어두운 선 | 딥 네이비 |
| 주조색 | 다크 틸, 바다색 청록 |
| 밝은 자수와 테두리 | 웜 아이보리 |
| 획득 조건의 핵심 강조 | 선명한 주황 |

- 주황은 한 뱃지 안에서 핵심 의미 한 가지에만 사용한다.
- 다른 계열의 강한 색을 새로 추가하지 않는다.
- 잠긴 뱃지는 별도 흑백 에셋을 만들지 않고 앱에서 투명도와 명도 처리를 적용한다.

## 상징 구성

- 뱃지 하나에는 주 상징 1개와 보조 상징 최대 2개만 사용한다.
- 같은 카테고리의 기존 뱃지와 중앙 상징이 겹치지 않게 한다.
- 횟수가 핵심인 뱃지는 점, 매듭, 항로 표식처럼 셀 수 있는 요소로 표현할 수 있다.
- 숫자 요소를 사용할 때는 해금 조건과 개수가 정확히 일치해야 한다.
- 배경 장식은 주 상징보다 명암 대비와 자수 밀도를 낮춘다.

## 에셋 규격

- 파일: `PNG`, `RGBA`, 투명 배경
- 캔버스: `512 × 512px`
- 뱃지의 시각적 점유율: 캔버스 너비와 높이의 약 82–88%
- 안전 여백: 모든 방향 최소 6%
- 외곽 실루엣의 중심과 이미지 중심을 맞춘다.
- 앞면과 뒷면은 같은 외곽 실루엣과 같은 시각적 크기를 사용한다.
- 파일명은 영문 소문자 케밥 표기법을 사용한다. 예: `night-sea-light.png`

## 검수 기준

1. 148px에서 중앙 상징과 카테고리를 구분할 수 있는가?
2. 외곽 형태가 카테고리 정책과 일치하는가?
3. 기존 팔레트와 자수 질감을 유지하는가?
4. 해금 조건을 나타내는 표식의 개수가 정확한가?
5. 같은 카테고리의 기존 뱃지와 실루엣 또는 상징이 지나치게 비슷하지 않은가?
6. 투명 배경, 여백, 중앙 정렬이 앞면과 뒷면에서 일치하는가?

## 신규 후보 적용

| 이름 | 조건 | 카테고리 | 형태 | 핵심 상징 |
| --- | --- | --- | --- | --- |
| 노련한 항해 | 출조 10회 완료 | `VOYAGE` | 방패형 | 선박 조타륜과 항로 표식 |
| 열 번의 손맛 | 조과 10건 기록 | `CATCH` | 원형 | 릴, 낚싯줄, 매듭 10개 |
| 바다 탐구자 | 어종 20종 수집 | `SPECIMEN` | 육각형 | 어종 표본과 확대경 |
| 기록의 주인 | 대어 기록 달성 | `CATCH` | 원형 | 대어, 줄자, 기록선 |
| 밤바다의 불빛 | 야간 출조 완료 | `FIELD NOTE` | 팔각형 | 달, 배, 빛나는 찌 |
| 다섯 개의 항로 | 낚시터 5곳 기록 | `VOYAGE` | 방패형 | 해도와 지점 5개 |
